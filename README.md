# Clone git and install environement

```sh
$ git clone https://github.com/mconvers/MathieuConvers_05_09112020
$ cd MathieuConvers_05_09112020/
$ yarn install
```
## Setup

### Boilerplate
For setup see the [boilerplate page](https://github.com/WeAreAthlon/frontend-webpack-boilerplate)


# Start the project

This project use this [back-end API](https://github.com/OpenClassrooms-Student-Center/JWDP5)

## With Boilerplate
```sh
$ yarn run serve
```
Go to http://127.0.0.1:8000/ in your browser

## Without Boilerplate / web server
Go to Orinico folder and open index.html in a browser.

Go to http://localhost:9999 in your browser
